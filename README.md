# Gemini Markdown Fixer

Gemini 웹사이트에서 발생하는 마크다운 렌더링 오류(볼드체와 한글 사이 공백 누락)를 해결하는 크롬 확장 프로그램입니다.

## 주요 기능
- **자동 감지**: MutationObserver를 통해 실시간으로 채팅 답변을 감지합니다.
- **마크다운 수정**: 볼드체(`**텍스트**`) 뒤에 바로 붙어 있는 한글 앞에 공백을 추가합니다.
- **보안 준수**: Gemini의 Trusted Types 정책을 준수하여 안전하게 DOM을 조작합니다.
- **무한 루프 방지**: DOM 수정 시 Observer를 일시적으로 중단하여 안정성을 확보합니다.

## 설치 방법
1. 이 리포지토리를 클론합니다.
2. 크롬 브라우저에서 `chrome://extensions/`에 접속합니다.
3. '개발자 모드'를 활성화합니다.
4. '압축해제된 확장 프로그램을 로드합니다' 버튼을 클릭하고 이 폴더를 선택합니다.
